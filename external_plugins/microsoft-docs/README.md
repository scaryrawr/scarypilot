# Microsoft Docs Plugin

Search and retrieve Microsoft Learn documentation directly from GitHub Copilot. Powered by the [Microsoft Learn MCP server](https://learn.microsoft.com/api/mcp).

## What This Plugin Does

Access Microsoft's official documentation through natural language:

- Search Microsoft Learn for articles, tutorials, and API references
- Get documentation for Azure, .NET, TypeScript, Power Platform, and more
- Retrieve code samples and best practices from official sources

## Installation

Install the plugin:

```bash
copilot plugin marketplace add scaryrawr/scarypilot
copilot plugin install microsoft-docs@scarypilot
```

## Usage Examples

**Search documentation:**

```text
"Find the Microsoft Learn docs on Azure Functions triggers"
"Look up the .NET 9 migration guide on Microsoft Learn"
```

**Get API references:**

```text
"Search Microsoft docs for the HttpClient class in .NET"
"Find the TypeScript handbook on Microsoft Learn"
```

**Explore tutorials:**

```text
"Find a tutorial for deploying a container app to Azure"
"Search Microsoft Learn for getting started with Blazor"
```
